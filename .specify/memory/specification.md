# Wish Pool - 產品規格

## 1. 概述

### 願景
建立一個溫暖的線上許願池平台，讓人們可以分享願望、給予祝福，並見證願望實現的美好時刻。透過社群的力量，讓每個願望都有被看見和實現的機會。

### 目標
- 提供簡單直觀的許願和祝福功能
- 建立正向互動的社群氛圍
- 視覺化呈現願望的聚集與實現
- 保護使用者隱私的同時促進真誠分享

### 非目標
- 不做商業化的願望交易平台
- 不涉及金錢往來或募資功能
- 不做社交媒體式的複雜互動
- 不收集過多個人資訊

## 2. 使用者故事

### 許願者
身為一個**許願者**，我要：
- 能夠簡單快速地發布我的願望
- 選擇願望的公開程度（公開/匿名）
- 為願望設定分類標籤
- 追蹤願望獲得的祝福數量
- 標記願望為「已實現」並分享喜悅

### 祝福者
身為一個**祝福者**，我要：
- 瀏覽不同類別的願望
- 為喜歡的願望送上祝福
- 留下鼓勵的話語
- 追蹤我祝福過的願望狀態
- 收到願望實現的通知

### 訪客
身為一個**訪客**，我要：
- 瀏覽公開的願望池
- 查看熱門願望和最新願望
- 了解平台的運作方式
- 快速註冊成為會員

## 3. 功能需求

### 3.1 願望管理
- **發布願望**
  - 願望標題（必填，50字內）
  - 願望描述（選填，500字內）
  - 分類標籤（健康、學業、事業、感情、其他）
  - 公開設定（公開/匿名）
  - 願望期限（選填）

- **願望展示**
  - 願望卡片式呈現
  - 顯示祝福數量
  - 顯示發布時間
  - 實現狀態標記

- **願望互動**
  - 一鍵送祝福（星星/愛心圖示）
  - 留言祝福功能
  - 分享願望連結
  - 舉報不當內容

### 3.2 使用者系統
- **註冊登入**
  - Email 註冊
  - Google/Facebook 快速登入
  - 暱稱設定
  - 頭像上傳（選填）

- **個人中心**
  - 我的願望列表
  - 我的祝福記錄
  - 通知中心
  - 個人資料編輯

### 3.3 社群功能
- **願望池首頁**
  - 熱門願望（依祝福數）
  - 最新願望
  - 即將到期的願望
  - 已實現的願望故事

- **分類瀏覽**
  - 依標籤分類
  - 搜尋功能
  - 排序選項（時間/熱門/即將到期）

### 3.4 通知系統
- 願望獲得祝福通知
- 願望獲得留言通知
- 祝福的願望實現通知
- 系統公告

## 4. 技術規格

### 架構
```
wish-pool/
├── .specify/
│   └── memory/
│       ├── constitution.md
│       └── specification.md
├── src/
│   ├── app/              # Next.js App Router
│   ├── components/       # React 元件
│   ├── lib/             # 工具函數
│   ├── hooks/           # Custom Hooks
│   ├── styles/          # 全域樣式
│   └── types/           # TypeScript 定義
├── public/              # 靜態資源
├── tests/              # 測試檔案
└── docs/               # 文檔

### 核心模組
- **WishModule**: 願望的建立、編輯、刪除
- **BlessingModule**: 祝福功能與計數
- **UserModule**: 使用者認證與管理
- **NotificationModule**: 通知推送與管理
- **AnalyticsModule**: 數據統計與分析

### 資料模型
```typescript
interface Wish {
  id: string;
  title: string;
  description?: string;
  category: Category;
  userId: string;
  isAnonymous: boolean;
  blessingCount: number;
  status: 'active' | 'fulfilled' | 'expired';
  deadline?: Date;
  createdAt: Date;
  updatedAt: Date;
}

interface Blessing {
  id: string;
  wishId: string;
  userId: string;
  message?: string;
  createdAt: Date;
}

interface User {
  id: string;
  email: string;
  nickname: string;
  avatar?: string;
  wishes: Wish[];
  blessings: Blessing[];
  createdAt: Date;
}
```

## 5. UI/UX

### 介面元件
- **WishCard**: 願望展示卡片
- **BlessingButton**: 祝福互動按鈕
- **CategoryFilter**: 分類篩選器
- **WishForm**: 許願表單
- **ProfileCard**: 使用者資料卡片

### 視覺設計
- **色彩方案**
  - 主色：柔和的藍紫漸變
  - 輔助色：暖黃色（祝福）、綠色（實現）
  - 中性色：灰階系統

- **動畫效果**
  - 願望卡片浮現動畫
  - 祝福星星飄散效果
  - 願望實現慶祝動畫

- **響應式斷點**
  - 手機：< 640px
  - 平板：640px - 1024px
  - 桌面：> 1024px

## 6. 成功指標

- **使用者參與**
  - 日活躍使用者(DAU): 1000+
  - 每日新增願望數: 100+
  - 平均每願望祝福數: 10+

- **使用者留存**
  - 次日留存率: > 40%
  - 七日留存率: > 20%
  - 月留存率: > 10%

- **平台健康度**
  - 願望實現率: > 30%
  - 不當內容比率: < 1%
  - 頁面載入時間: < 2秒
